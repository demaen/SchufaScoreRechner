# Schufa-Score-Rechner

Kurzbeschreibung
- Dieses kleine Tool zeigt beispielhaft, wie ein Schufa-Score berechnet werden kann. Es dient nur als Referenz und ist kein offizielles Produkt der SCHUFA.

Berechnungsgrundlage
- Die Darstellung und die Berechnungslogik beruhen nach bestem Wissen und Gewissen auf Informationen von https://www.schufa.de/scoring-daten/neuer-score/index.jsp (Stand 17.03.2026).

Live-Demo
- Die App ist erreichbar unter: https://demaen.github.io/SchufaScoreRechner/

Nutzung
- Lokal: `index.html` im Browser öffnen.
- Online: Die oben genannte Demo-Seite besuchen.

Haftungsausschluss
- Es wird keine Gewähr für Vollständigkeit, Richtigkeit oder Aktualität übernommen.
- Der tatsächlich gültige Score wird ausschließlich von der SCHUFA ermittelt. Diese Anwendung liefert lediglich eine Orientierung und ersetzt keine offizielle Auskunft.

Kontakt / Fehler
